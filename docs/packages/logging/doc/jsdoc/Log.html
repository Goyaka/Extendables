Class Log¶

A log file object. Will resume an existing log if it exists, or make a new one if it doesn’t.

Examples¶

 var logging = require("logging");
 // all logs end up in extendables/log
 var log = new Log("example.log");
 try {
     throw new Error();
 } catch (error) {
     log.debug("Caught error, will log a debug message now");
     log.critical("Something happened: {error} ({env})", {'error': error, 'env': $.os});
}

Constructor¶

class Log(name[, log_level])¶
Arguments:
  • name (String) – the filename for this log file
  • log_level (String|Number) – the log level above which log messages don’t get written,

either a name (e.g. CRITICAL) or the actual log level. (Default: 4)
