Class UI¶

This is the base class from which Dialog and Palette inherit. Don’t instantiate this class directly.

el¶

UI#el()¶

To make UI methods chainable, they return the entire UI object instead of any control element you may have just created; obj.el() gives you easy access to the last added element.

var dialog = new ui.Dialog("Hello");
dialog.button('ok', 'Click here').el().size = [400, 200];
