Class Log
================================================================================

..
   class-title


A log file object. Will resume an existing log if it exists, 
or make a new one if it doesn't.








    

Examples
--------


.. code-block:: javascript

    var logging = require("logging");
    // all logs end up in extendables/log
    var log = new Log("example.log");
    try {
        throw new Error();
    } catch (error) {
        log.debug("Caught error, will log a debug message now");
        log.critical("Something happened: {error} ({env})", {'error': error, 'env': $.os});
   }



Constructor
-----------

.. js:class:: Log(name, [log_level])


    
    :param String name: 
        the filename for this log file 
    
    :param String|Number log_level: 
        the log level above which log messages don't get written, 
either a name (e.g. CRITICAL) or the actual log level. (*Default*: 4)
