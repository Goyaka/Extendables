String methods¶

Extendables provides a number of additional string munging and manipulation methods for String objects. Most of these will look familiar if you’ve coded in other languages before.

contains¶

String#contains(substring)¶
Arguments:
  • substring (String) –
Returns Bool:

True or false.

Tests whether the string contains the specified substring. This is equal to str.indexOf(substring) != -1.

endswith¶

String#endswith(substring)¶
Arguments:
  • substring (String) –
Returns Bool:

True or false.

Tests whether the string ends with the specified substring.

format¶

String#format(replacements)¶
Arguments:
  • replacements (String) – For each {} symbol in the text, format expects a replacement argument.

Calls .toString() on each replacement, so you can pass in any data type. You may also specify a single replacement object, which will do named formatting.

This is a simple string formatting method, loosely inspired on the one in Python 3.

  • In unnamed mode, specify placeholders with the {} symbol.
  • In named mode, specify placeholders with {propname}.
> var person = {'salutation': 'mister', 'name': 'John Smith'};
> var hello = "Hello there, {}, I've heard your name is {}!".format(person.salutation, person.name);
> $.writeln(hello);
"Hello there, mister, I've heard your name is John Smith"

> var person = {'salutation': 'mister', 'name': 'John Smith'};
> var hello = "Hello there, {salutation}, I've heard your name is {name}!".format(person);
> $.writeln(hello);
"Hello there, mister, I've heard your name is John Smith"

indexAfter¶

String#indexAfter(substring)¶
Arguments:
  • substring –

Does what it says. Does not check whether the string actually extends beyond the the substring.

ltrim¶

String#ltrim([character])¶
Arguments:
  • character (String) – if specified, removes leading characters matching the parameter

instead of whitespace.

Removes leading whitespace characters, including tabs, line endings and the like.

> $.writeln("   hello there   ".trim());
"hello there   "

rtrim¶

String#rtrim([character])¶
Arguments:
  • character (String) – if specified, removes trailing characters matching the parameter

instead of whitespace.

Removes trailing whitespace characters, including tabs, line endings and the like.

> $.writeln("   hello there   ".trim());
"   hello there"

startswith¶

String#startswith(substring)¶
Arguments:
  • substring (String) –
Returns Bool:

True or false.

Tests whether the string starts with the specified substring.

trim¶

String#trim([character])¶
Arguments:
  • character (String) – if specified, removes leading and trailing characters matching the

parameter instead of whitespace.

Removes leading and trailing whitespace characters, including tabs, line endings and the like.

> $.writeln("   hello there   ".trim());
"hello there"
