Array methods¶

Extendables adds a considerable amount of methods to the Array object prototype. Most of these will be familiar from either Javascript 1.6+ (modern browsers implement Javascript 1.8 or higher) or from Jeremy Ashkenas’ underscore.js library.

// select all people who are statesmen and give
// them an especially warm welcome
people.select(function (person) {
    return person.statesman;
}).forEach(function (person) {
    if (person.has('title')) {
        var title = person.title;
    } else {
        var title = 'Mr.';
    }
    alert("And I'd especially like to welcome {} {}".format(title, person.last));
});

compact¶

Array#compact()¶

Returns a copy of the array with all falsy values removed. This includes false, null, 0, "", undefined and NaN.

contains¶

Array#contains(obj)¶
Arguments:
  • obj –

Similar to indexOf

every¶

Array#every(function)¶
Arguments:
  • function (Function) –

Tests whether all elements in the array pass the test implemented by the provided function.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/every

filter¶

Array#filter(fun)¶
Arguments:
  • fun –

Creates a new array with all elements that pass the test implemented by the provided function.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/filter

first¶

Array#first()¶

Returns the first item of this array

flatten¶

Array#flatten()¶

Flattens nested arrays.

> var list = [[1, 2, [3, 4, 5], 6], [7, 8], 9];
> list.flatten();
[1,2,3,4,5,6,7,8,9];

forEach¶

Array#forEach(function)¶
Arguments:
  • function (Function) –

Executes a provided function once per array element.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/forEach

indexAfter¶

Array#indexAfter(element)¶
Arguments:
  • element –

Does what it says. Does not check whether there is actually a next element, that’s up to you.

indexOf¶

Array#indexOf(element)¶
Arguments:
  • element (Object) –

Returns the first index at which a given element can be found in the array, or -1 if it is not present.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/indexOf

last¶

Array#last()¶

Returns the last item of this array

lastIndexOf¶

Array#lastIndexOf(element, from_index)¶
Arguments:
  • element (Object) –
  • from_index (Number) –

Returns the last index at which a given element can be found in the array, or -1 if it is not present. The array is searched backwards, starting at from_index.

See also

https://developer.mozilla.org/en/Core_JavaScript_1.5_Reference/Global_Objects/Array/lastIndexOf

map¶

Array#map(function)¶
Arguments:
  • function (Function) –

Creates a new array with the results of calling a provided function on every element in this array.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/map

max¶

Array#max([salient_feature])¶
Arguments:
  • salient_feature (Function|String) – min can also order objects

if you provide a salient feature for it to work on, either a function or the name of an object property

Returns the maximum value in an array.

> var people = [{'name': 'Alfred'}, {'name': 'Zed'}];
> people.max(function (obj) {
... return obj.name;
... });
{'name': 'Zed'}

min¶

Array#min([salient_feature])¶
Arguments:
  • salient_feature (Function|String) – See max.
:returns :
The minimum value in an array. Works like Array#max()

pluck¶

Array#pluck(name)¶
Arguments:
  • name –

Allows you to quickly pluck a single attribute from an array of objects.

> var people = [{'name': 'Alfred', age: 33}, {'name': 'Zed', age: 45}];
> people.pluck('age');
[33,45]
> people.pluck('age').sum();
78
> people.sum('age');
78
> people.sum(function (person) { return person.age });
78

reduce¶

Array#reduce(function)¶
Arguments:
  • function (Function) –

Apply a function against an accumulator and each value of the array (from left-to-right) as to reduce it to a single value.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/Reduce

reduceRight¶

Array#reduceRight(function)¶
Arguments:
  • function (Function) –

Apply a function simultaneously against two values of the array (from right-to-left) as to reduce it to a single value.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/ReduceRight

reject¶

Array#reject(fn)¶
Arguments:
  • fn (Function) –

Does exactly the inverse of Array#filter() and its alias Array#select().

select¶

Array#select()¶

Alias for Array#filter()

some¶

Array#some(function)¶
Arguments:
  • function (Function) –

Tests whether some element in the array passes the test implemented by the provided function.

See also

https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/some

sum¶

Array#sum([salient_feature])¶
Arguments:
  • salient_feature (Function|String) – See max.
:returns :
The sum of all array values.
> var persons = [
... {'name': 'Abraham', 'children': 5},
... {'name': 'Joe', 'children': 3},
... {'name': 'Zed', 'children': 0}
... ];
> persons.sum('children');
8
